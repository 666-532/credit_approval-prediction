# Credit Approval Prediction using Machine Learning

This project builds a machine learning model to predict whether a credit application should be approved or rejected.

## Dataset
The dataset is taken from the UCI Machine Learning Repository.

## Project Steps
1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Encoding Categorical Variables
5. Feature Scaling
6. Model Training
7. Model Evaluation

## Models Used
- Logistic Regression

## Results
Training Accuracy: 87%  
Testing Accuracy: 85%

## Evaluation Metrics
- Confusion Matrix
- Precision
- Recall
- F1 Score
- ROC Curve
